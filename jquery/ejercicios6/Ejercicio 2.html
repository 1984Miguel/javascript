Entorno Cliente

JavaScript

JavaScript es un lenguaje de programación interpretado en el lado del cliente.

DOM

El DOM es la traducción que realiza el navegador del documento HTML a objetos accesibles mediante JavaScript.

jQuery

jQuery es una librería de JavaScript que facilita el desarrollo de sitios web complejos

Ajax

Ajax es un conjunto de técnicas que permite intercambiar información entre el documento en el cliente y el servidor sin recargar el documento.
